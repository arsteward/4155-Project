Objective


There is so much information available at our fingertips. In this digital age, it is often difficult and time-consuming to determine if a body of text is what we are looking for. With this application there is an efficient way to generate a concise summary of any text, text file or image!

Sometimes all we need is a brief overview. Summaries are easier to understand than long-winded texts filled with scientific jargon. Generating a summary with all the main points is much faster than trying to skim through an entire document and comprehend the important parts. The aim of this application help users decide if they would like to pass on a document or read deeper into it without having to spend the time to skim or read the entire text.
