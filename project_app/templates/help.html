Help


How to use the application



Hover on the Menu Icon in the top right and click "Summarize" or click the "Go to Text Summarization" button below.

Copy + Paste text into the text box or choose a .png or .txt file from your computer.

Select an algorithm from the algorithms drop down menu(See algorithm description table below)

Select the length of the summary from the sentences drop down menu.

Click submit: This will lead to a summary page containing the summary and the original text.

Algorithms


Distance: Selects the longest sentence in the text. If multiple sentences are requested it will display in descending order of length.

Bag of words: Uses histograms to find the sentences that best resemble the overall text

Pre-trained Transformers: Uses open-source transformer models for abstractive summary generation. These models have hundreds of millions of parameters, so generation may take some time
