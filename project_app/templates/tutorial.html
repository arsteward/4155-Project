Tutorial


Copy + Paste text into the text box, choose an image file or choose a .txt file from your computer. (For more information on which formats are available for image and text uploads check out the help page or click the information icon)

Select an algorithm from the algorithms drop down menu (For help understanding the different algorithms check out the algorithm description table in the Help page from the navigation menu in the top right)

Select the length of the summary from the sentences drop down menu.

Press submit.

Once submit is pressed, you will be navigated to the summary page containing your summary and original text

Click the button below to get started!
