Tutorial


Copy + Paste text into the text box, choose an image file or choose a .txt file from your computer.

Select an algorithm from the algorithms drop down menu (See algorithm description table in the Help Tab from the navigation menu in the top right)

Select the length of the summary from the sentences drop down menu. Then press submit.

Once submit is pressed, you will be navigated to the summary page containing your summary and original text
